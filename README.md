<h3>How to run [DEV]</h3>
<ol type="1">
  <li>download and unzip the file</li>
  <li>open two consoles, one for frontend folder and one for backend folder</li>
  <li>use npm install to get all the necessary libraries (and check if you need to install additional node.js and vita)</li>
  <li>after installing, use npm run dev in both consoles. Then use the localhost from the frontend console to access the webpage</li>
  <li> Install all the libraries and elements for MySQL and Mongoose library management </li>
  <li> In MySQL, after creating  new connection with the .env data, import the database file from the DATABASE folder.<li>
</ol>

<h3>main reference for the API functions</h3>
<ol type="1">
  <li>api/leaderboard/ shows the leaderboard</li>
  <li>api/connect/ shows the </li>
  <li>api/login/ shows all the users</li>
  <li>api/login/:id finds a user by _id</li>
  <li>api/login/users/:username/ finds a user by username </li>
  <li>api/login/email/:email/ finds a user by its email<li>
</ol>
